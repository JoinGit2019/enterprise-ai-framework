# ApiDesign

## 说明

设计接口定义、请求响应结构、错误码和调用方式。

## 适用场景

- 新增接口
- 修改接口返回
- 补充接口文档
- 对接第三方服务

## 输入

- 需求
- 业务流程
- 现有接口约束
- 鉴权规则

## 输出

- 接口路径
- 请求参数
- 响应结构
- 错误码
- 示例

## 关键规则

- 接口语义必须清晰
- 返回结构必须稳定
- 错误码必须可追踪

## 必填项

- 接口路径
- 请求参数
- 响应结构
- 错误码
- 示例

## 输出要求

- 必须说明请求和返回的约束
- 必须给出可理解的错误码
- 必须让后续 ApiDocumentation 可以直接引用
