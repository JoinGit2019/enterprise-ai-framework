# DatabaseDesign

## 说明

输出数据库设计与变更说明。

## 适用场景

- 新建数据表
- 修改字段
- 调整索引
- 数据迁移

## 输入

- 需求
- 技术方案
- 现有表结构
- 数据规模

## 输出

- 表结构建议
- 字段说明
- 索引建议
- 迁移方案
- 回滚方案

## 关键规则

- 设计必须考虑查询模式
- 索引必须与业务场景匹配
- 迁移必须包含回滚路径

## 必填项

- 表结构建议
- 字段说明
- 索引建议
- 迁移方案
- 回滚方案

## 输出要求

- 必须考虑查询与写入模式
- 必须说明迁移影响
- 必须提供回滚路径
