# ChangeSummary

## 说明

总结本次改动、影响面和后续注意事项。

## 适用场景

- 提交代码后
- 发布前
- 接口或行为变更后

## 输入

- 代码变更
- 测试结果
- 接口变更
- 风险说明

## 输出

- 变更摘要
- 影响范围
- 回滚提示
- 后续跟进项

## 关键规则

- 说明改了什么
- 说明影响了什么
- 说明还需要注意什么

## 必填项

- 变更摘要
- 影响范围
- 回滚提示
- 后续事项

## 输出要求

- 必须说明影响面
- 必须说明回滚方式
- 必须说明后续跟进项
