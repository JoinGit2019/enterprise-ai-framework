# Templates

## 说明

Templates 定义每类交付文档的标准结构，保证不同 Workflow 产出的文档具有一致的阅读方式和可追溯性。

## 当前模板

- `RequirementAnalysis`
- `TechnicalDesign`
- `RiskAnalysis`
- `ExistingLogicAnalysis`
- `ApiDocumentation`
- `ChangeSummary`
- `ReleaseNote`

## 使用原则

- 模板是文档骨架，不是工作流
- 模板必须简洁、稳定、可复用
- 模板内容应与 Workflow 输出一致
- 模板应支持中文写作习惯
