# HTML-CSS-JavaScript

## 说明

适用于不依赖特定前端框架的通用前端项目，或业务项目中的 HTML、CSS、JavaScript、模板页面和组件页面修改。

- `Runtimes/Frontend/CSS`
- `Runtimes/Frontend/JavaScript`

## 优先级

1. 项目现有页面、组件和样式体系
2. 浏览器目标和构建环境
3. HTML / CSS / JavaScript 兼容要求
4. 前端通用规则
5. 企业研发标准

## 使用原则

- 优先复用已有页面、组件、样式和工具函数
- 不随意引入前端框架、插件或构建工具
- 样式优先写在 CSS、样式文件或项目既有样式体系中
- 修改 JS 时必须确认事件绑定、接口参数和返回结构
- 不使用当前项目构建环境不支持的语法
